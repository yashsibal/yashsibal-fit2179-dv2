# How Australia Moves — FIT2179 Data Visualisation 2

This repository contains a single-page Vega-Lite data visualisation story for FIT2179 Data Visualisation 2.

## Topic

**Australia’s changing sport, fitness and physical activity habits.**

The story asks whether adult participation is mainly shaped by traditional organised sport or by flexible lifestyle activities such as walking, gym/fitness, running and swimming.

## Published site

The final website is intended to be hosted using GitHub Pages from the repository root.

## Files

- `index.html` — single-page scrolling visualisation story
- `style.css` — layout, typography and responsive styling
- `script.js` — embedded Vega-Lite chart specifications and rendering logic
- `js/` — human-readable Vega-Lite JSON specifications for each visualisation
- `data/` — cleaned CSV data used to create the visualisations
- `images/` — sketch image/PDF location
- `.nojekyll` — ensures GitHub Pages serves the static files normally

## Data sources

1. Australian Sports Commission, AusPlay Data Tables, January–December 2025.
2. Australian Bureau of Statistics, National, state and territory population, September 2025.

## Visualisation requirements addressed

- Single public webpage
- Vega-Lite diagrams/maps
- 12 visualisations
- Geographic map section
- Multiple idioms: bar ranking, stacked bars, heatmap, projected map, dot ranking, scatterplot, diverging bars, small multiples and custom spectrum
- What / Why / Who / How framework
- Source notes, authorship and AI acknowledgement
