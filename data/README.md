# Data folder

These CSV files contain cleaned values extracted from the official AusPlay and ABS workbooks uploaded for the project.

## Main sources

- Australian Sports Commission AusPlay Data Tables, January–December 2025.
- Australian Bureau of Statistics National, state and territory population, September 2025.

## Interpretation note

AusPlay participation rates are estimates. Adults can participate in more than one activity, so activity percentages should not be added together as if they represent unique people.
